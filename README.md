# ⚽ Dugout 2k26

Simulazione manageriale calcistica direttamente dal browser.
Crea la tua carriera, gestisci la squadra e porta il tuo club alla vittoria 🏆

---

## 🚀 Gioca subito

👉 https://Dugout2k26.github.io

---

## 🎮 Funzionalità

* ⚽ Simulazione partite
* 🔄 Squadre e giocatori generati casualmente
* 💰 Sistema di mercato
* 📊 Gestione squadra
* 🧠 Logica di gioco in JavaScript
* 🌐 Giocabile direttamente online (nessun download)

---

## 🛠️ Tecnologie utilizzate

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## 📦 Come funziona

Il gioco gira completamente nel browser:

* Nessun server richiesto
* Tutto gestito tramite JavaScript
* Salvataggi locali (in futuro)

---

## 📁 Struttura progetto

```
index.html   → gioco completo
README.md    → descrizione progetto
```

---

## 🔧 Installazione (opzionale)

Se vuoi modificarlo:

1. Clona il repository:

```
git clone https://github.com/Dugout2k26/Dugout2k26.github.io
```

2. Apri `index.html` nel browser
